# **App Name**: AI Tool Scout

## Core Features:

- AI Tool Browsing & Search: Allows users to search for AI tools by name and filter them by profession, work type, pricing model, and popularity using Firestore data.
- AI Tool Detail Pages: Displays comprehensive information for each AI tool, including its description, features, pros, cons, pricing, and links, pulled from Firestore.
- Tool Comparison Engine: Enables users to select 2-4 AI tools and compare their key attributes side-by-side in a dynamic table, utilizing Firestore for data retrieval.
- AI-Enhanced SEO Description Tool: Utilizes a generative AI tool to assist in creating or optimizing detailed, SEO-friendly descriptions (minimum 800 words) and meta-data for new tool submissions and existing tool pages, which are then saved to Firestore.
- AI Tool Submission & Moderation: Provides a submission form for users to add new AI tools, which are stored in Firestore with an 'approved' status awaiting admin review.
- Admin Content Management System: A secure admin panel to manage AI tool submissions (approve/reject), edit tool details, feature tools on the homepage, and organize categories directly in Firestore.
- Monetization Ad Placements: Integrates designated placeholder areas across the website for Google AdSense banners, enabling monetization through advertising.

## Style Guidelines:

- Primary interactive elements color: A sophisticated indigo hue (#4F38AA) representing intelligence and innovation, providing a sense of depth and modernity for a 'Modern SaaS' style application.
- Background color: A subtle, desaturated lavender-grey (#F5F3F8), visually aligned with the primary color while ensuring a clean and airy feel that contrasts effectively with text and interactive elements.
- Accent color: A vibrant, clear blue (#3B5AFF) used sparingly for call-to-action buttons, highlights, and other attention-grabbing elements, ensuring visual pop and excellent contrast against both primary and background colors.
- Headline and body font: 'Inter', a contemporary grotesque sans-serif that offers excellent readability and a modern, neutral aesthetic suitable for a tech-focused, 'Clean typography' application.
- Utilize modern, clean line icons for navigation, tool features, and comparison points, maintaining visual consistency with the SaaS-inspired interface.
- Implement a card-based layout for tool listings and detail sections, prioritizing mobile-first responsiveness and intuitive visual grouping of information.
- Incorporate subtle, performant animations for state changes, hovers, and transitions, enhancing user experience without compromising 'fast loading' performance.
